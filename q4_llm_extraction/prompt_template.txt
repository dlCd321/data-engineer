================================================================
Q4 LLM Prompt 模板
================================================================
请在此处设计你的 Prompt。这是评分的重要部分。

设计建议：
1. System Prompt + Few-shot examples 通常优于纯 Zero-shot
2. 明确规定输出 JSON schema
3. 加入"如果原文没提到，请不要编造"这类反幻觉指令
4. 葡萄牙语处理：让模型直接处理 vs 先翻译再处理
5. 要求 evidence_quote 必须出自原文（防幻觉）

# === Few-shot Examples（如有）===

Example 1:
Input: "Produto chegou quebrado, embalagem rasgada. Pessimo!"
Expected Output:
{
  "issues": [
    {
      "category": "quality_defect",
      "severity": "high",
      "evidence_quote": "Produto chegou quebrado",
      "evidence_quote_en": "Product arrived broken"
    },
    {
      "category": "packaging",
      "severity": "medium",
      "evidence_quote": "embalagem rasgada",
      "evidence_quote_en": "packaging torn"
    }
  ],
  "sentiment_score": -0.9,
  "actionable_for_seller": true,
  "confidence": 0.95
}


# === Output Schema ===

{
  "review_id": "...",
  "issues": [
    {
      "category": "delivery_late | wrong_product | quality_defect | missing_item | seller_communication | packaging | other",
      "severity": "low | medium | high",
      "evidence_quote": "原文中证明这个问题的句子（葡萄牙语）",
      "evidence_quote_en": "英文翻译"
    }
  ],
  "sentiment_score": -1.0 ~ 1.0,
  "actionable_for_seller": true | false,
  "confidence": 0.0 ~ 1.0
}
